# Error Log — 누적 오류 기록 및 학습

이 파일은 시스템 운영 중 발생한 오류와 그 해결 과정을 누적 기록한다.
에이전트는 P0/P1 오류 해결 후 반드시 이 파일을 업데이트해야 한다.

---

## 기록 형식

```
### [날짜] [오류등급] 제목
- **발생 위치**: 도구/워크플로우 경로
- **에러 메시지**: 전체 메시지 (축약 금지)
- **근본 원인**: 왜 발생했는가
- **수정 내용**: 어떤 코드/설정을 변경했는가
- **검증 방법**: 수정 후 어떻게 확인했는가
- **워크플로우 반영**: 어떤 워크플로우를 업데이트했는가
- **재발 방지**: 같은 오류가 다시 발생하지 않으려면 무엇이 필요한가
```
