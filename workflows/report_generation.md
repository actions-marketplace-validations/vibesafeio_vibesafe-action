# Workflow: 보안 리포트 생성 (report_generation)

## 목표
스캔 결과를 바이브 코더가 이해할 수 있는 보안 리포트로 변환한다.

## 리포트 구성 요소

### 1. 보안 점수 요약 (Security Score Summary)
- 0-100점 원형 게이지
- 등급: A (90-100), B (75-89), C (60-74), D (40-59), F (0-39)
- 이전 스캔 대비 변화량 (↑↓)

### 2. 도메인별 규제 준수 현황
- 해당 도메인에 적용되는 규제 목록 (PCI DSS, GDPR, HIPAA 등)
- 각 규제별 준수/미준수 항목 수
- 미준수 시 예상 리스크 (벌금, 서비스 중단 등)

### 3. 취약점 목록 (우선순위순)
각 취약점마다:
- 심각도 뱃지 (CRITICAL/HIGH/MEDIUM/LOW)
- 한줄 요약 (비전문가 언어)
- 상세 설명 (기술적)
- 영향 받는 파일과 라인 번호
- 수정 방법 (원클릭 패치 버튼 + AI 프롬프트 복사 버튼)

### 4. 의존성 취약점
- 취약 패키지 목록
- CVE ID 및 공개 날짜
- 안전한 버전 업그레이드 경로

### 5. 시크릿 노출
- 발견된 시크릿 유형과 위치
- 각 시크릿의 즉시 무효화 가이드

## 리포트 형식
| 형식 | 용도 |
|------|------|
| 웹 대시보드 (HTML) | 기본 형식, 인터랙티브 |
| PDF | 다운로드, 공유용 |
| JSON | API 연동, CI/CD 통합용 |

## 실행
```bash
python tools/report/pdf_generator.py \
  --scan-id <scan_id> \
  --format pdf \
  --language ko \
  --output .tmp/scan_<scan_id>/report.pdf
```
